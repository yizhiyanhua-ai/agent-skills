# 照片整理大师

智能整理照片库，按时间和内容自动分类，支持 EXIF 元数据提取、内容识别、重复检测、批量重命名。

## 核心功能

### 1. EXIF 元数据提取
- **拍摄时间**：从 EXIF 数据提取精确拍摄时间
- **拍摄地点**：GPS 坐标转换为地理位置
- **设备信息**：相机型号、镜头参数
- **照片参数**：分辨率、ISO、光圈、快门速度

### 2. 智能分类
基于时间和内容自动分类照片：
- **按时间**：年份/月份/日期
- **按内容**：风景、人物、美食、文档、截图
- **按事件**：旅行、聚会、工作、日常
- **按地点**：基于 GPS 数据的地理位置

### 3. 重复检测
- **完全重复**：MD5 哈希值相同
- **相似照片**：感知哈希（pHash）算法
- **连拍照片**：时间戳相近的照片组
- **编辑版本**：原图与编辑后的照片

### 4. 质量评估
- **模糊检测**：拉普拉斯方差算法
- **曝光检测**：过曝/欠曝分析
- **构图评估**：三分法、黄金分割
- **人脸检测**：人脸数量和清晰度

### 5. 批量操作
- **重命名**：按规则批量重命名（时间戳、序号）
- **移动**：按分类移动到目标文件夹
- **复制**：保留原文件，复制到新位置
- **删除**：移动到待删除文件夹（可恢复）

## 分类规则

### 按时间分类

**目录结构**：
```
照片库/
├── 2024/
│   ├── 01-一月/
│   ├── 02-二月/
│   └── ...
├── 2025/
│   ├── 01-一月/
│   ├── 02-二月/
│   └── ...
└── 2026/
    ├── 01-一月/
    ├── 02-二月/
    └── ...
```

### 按内容分类

**分类标准**：
- **风景**：自然景观、城市风光、建筑
- **人物**：肖像、合影、自拍
- **美食**：餐厅、家常菜、甜点
- **文档**：证件、票据、文字截图
- **截图**：手机/电脑截图
- **其他**：无法分类的照片

### 按事件分类

**事件识别**：
- **旅行**：连续多天、不同地点、风景照为主
- **聚会**：同一天、多人合影、室内场景
- **工作**：工作日、文档/截图为主
- **日常**：零散照片、无明显主题

## 重复检测算法

### 1. 完全重复（MD5）

```python
import hashlib

def calculate_md5(file_path):
    with open(file_path, 'rb') as f:
        return hashlib.md5(f.read()).hexdigest()
```

### 2. 相似照片（pHash）

```python
import imagehash
from PIL import Image

def calculate_phash(file_path):
    img = Image.open(file_path)
    return imagehash.phash(img)

def is_similar(hash1, hash2, threshold=5):
    return hash1 - hash2 < threshold
```

### 3. 连拍照片

时间戳相差小于 2 秒的照片视为连拍组。

## 质量评估

### 模糊检测

使用拉普拉斯方差算法：
```python
import cv2

def is_blurry(image_path, threshold=100):
    img = cv2.imread(image_path, cv2.IMREAD_GRAYSCALE)
    laplacian_var = cv2.Laplacian(img, cv2.CV_64F).var()
    return laplacian_var < threshold
```

### 曝光检测

分析直方图分布：
- **过曝**：高亮区域占比 > 30%
- **欠曝**：暗部区域占比 > 30%
- **正常**：中间调分布均匀

## 常见问题

### Q1：支持哪些图片格式？
A：JPG、PNG、HEIC、GIF、BMP、TIFF、RAW（CR2/NEF/ARW）

### Q2：会修改原始照片吗？
A：不会。仅移动/重命名文件，不修改照片内容和 EXIF 数据。

### Q3：如何处理无 EXIF 的照片？
A：使用文件修改时间作为备选，或移至"未分类"文件夹。

### Q4：重复检测会误删吗？
A：完全重复（MD5 相同）可安全删除。相似照片需人工确认。

### Q5：整理速度如何？
A：约 1000 张/分钟（仅 EXIF 提取），内容识别较慢（约 100 张/分钟）。

### Q6：支持视频文件吗？
A：支持提取视频 EXIF，但不支持内容识别。

### Q7：如何恢复误删的照片？
A：从"待删除"文件夹恢复，或使用操作日志撤销。

### Q8：可以自定义分类规则吗？
A：可以。编辑 `references/classification_rules.md` 自定义规则。
