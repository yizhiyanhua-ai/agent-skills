# 家庭账本管家

智能记录和分析家庭开支，支持多种输入方式（文本、语音、票据照片），自动分类记账，生成月度报告和趋势分析。

## 核心功能

### 1. 多模态输入
- **文本输入**：直接描述消费记录（如"今天午餐花了35元"）
- **语音输入**：语音描述消费，自动转录并记账
- **票据照片**：上传超市小票、发票照片，OCR 识别金额和商品

### 2. 智能分类
基于关键词和场景自动分类消费记录：
- 餐饮：餐厅、外卖、超市食品
- 交通：打车、公交、地铁、加油
- 购物：服装、日用品、电子产品
- 娱乐：电影、游戏、旅游
- 教育：培训、书籍、课程
- 医疗：药品、挂号、体检
- 住房：房租、物业、水电
- 其他：未分类项目

### 3. 数据管理
- **Excel 存储**：使用 Excel 文件持久化数据
- **字段完整**：日期、金额、类别、备注、支付方式
- **数据校验**：金额格式、日期有效性检查

### 4. 统计分析
- **月度汇总**：按月统计总支出、各类别支出
- **类别占比**：饼图展示各类别消费占比
- **趋势分析**：同比、环比增长率
- **预算管理**：设置预算上限，超支提醒

## 工作流程

### 流程 1：文本/语音记账

```
用户输入
  ↓
提取关键信息（金额、类别、日期、备注）
  ↓
智能分类（基于关键词匹配）
  ↓
数据校验（金额格式、日期有效性）
  ↓
写入 Excel 文件
  ↓
返回记账确认
```

**示例对话**：
```
用户：今天打车花了50块
助手：✅ 已记账
- 日期：2026-02-09
- 金额：¥50.00
- 类别：交通
- 备注：打车
- 当月交通支出：¥320.00
```

### 流程 2：票据照片记账

```
用户上传照片
  ↓
OCR 识别（Tesseract）
  ↓
提取金额、商品、日期
  ↓
智能分类（基于商品名称）
  ↓
用户确认（可修改）
  ↓
写入 Excel 文件
  ↓
返回记账确认
```

**示例对话**：
```
用户：[上传超市小票照片]
助手：识别到以下消费记录：
- 商品：牛奶、面包、鸡蛋
- 金额：¥68.50
- 日期：2026-02-09
- 建议分类：餐饮

是否确认记账？（可修改类别或备注）
```

### 流程 3：生成月度报告

```
用户请求报告
  ↓
读取 Excel 数据
  ↓
筛选指定月份数据
  ↓
统计分析（总支出、类别占比、趋势）
  ↓
生成 Markdown 报告
  ↓
可选：生成图表（饼图、折线图）
```

**示例对话**：
```
用户：生成本月消费报告
助手：# 2026年2月消费报告

## 总览
- 总支出：¥4,235.00
- 记账笔数：47笔
- 日均支出：¥141.17

## 类别分布
| 类别 | 金额 | 占比 | 笔数 |
|------|------|------|------|
| 餐饮 | ¥1,580.00 | 37.3% | 23 |
| 交通 | ¥520.00 | 12.3% | 8 |
| 购物 | ¥1,200.00 | 28.3% | 6 |
| 娱乐 | ¥450.00 | 10.6% | 4 |
| 其他 | ¥485.00 | 11.5% | 6 |

## 趋势分析
- 环比上月：+8.5%
- 超支类别：餐饮（预算¥1,200，实际¥1,580）
```

## 智能分类规则

### 关键词匹配表

| 类别 | 关键词 |
|------|--------|
| 餐饮 | 餐厅、外卖、超市、菜市场、水果、零食、饮料、咖啡、奶茶 |
| 交通 | 打车、滴滴、出租车、公交、地铁、加油、停车、高速 |
| 购物 | 淘宝、京东、拼多多、衣服、鞋子、化妆品、电子产品、家电 |
| 娱乐 | 电影、KTV、游戏、旅游、景点、酒店、健身 |
| 教育 | 培训、课程、书籍、学费、考试、证书 |
| 医疗 | 药店、医院、挂号、体检、药品、保健品 |
| 住房 | 房租、物业、水费、电费、燃气、宽带、维修 |

### 分类优先级
1. **精确匹配**：如"滴滴打车" → 交通
2. **模糊匹配**：如"买菜" → 餐饮
3. **金额推断**：如大额支出 + "房" → 住房
4. **用户历史**：学习用户习惯（如"711" → 餐饮）

```

```

```
## 总支出趋势

- 2月：¥4,235.00（+2.8%）

## 类别变化

- 交通：基本稳定
```

- **Python 脚本**：

```bash
pip install openpyxl  # Excel 操作
```

## 常见问题

### Q1：如何修改已记录的消费？
A：说"修改最近一笔记录"或"修改2月9日的打车费用"，我会帮你定位并修改。

### Q2：支持多人共用一个账本吗？
A：支持。可以在备注中标注消费人（如"张三-午餐"），报告时可按人统计。

### Q3：如何导出数据？
A：Excel 文件可直接打开编辑，也可以导出为 CSV 格式用于其他工具分析。

### Q4：OCR 识别不准确怎么办？
A：识别后会让你确认，可以手动修改金额和类别。建议拍照时保持票据清晰、光线充足。

### Q5：如何设置预算？
A：说"设置餐饮预算为1200元"，我会更新预算设置表。超支时会自动提醒。

### Q6：支持哪些支付方式？
A：支持微信、支付宝、现金、银行卡等，可在记账时指定或后续补充。

### Q7：如何查看历史记录？
A：说"查看2月的所有餐饮支出"或"显示最近10笔记录"，我会从 Excel 中筛选并展示。

### Q8：数据安全吗？
A：数据存储在本地 Excel 文件，不会上传到云端。建议定期备份到云盘或移动硬盘。
