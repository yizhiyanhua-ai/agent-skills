# 会议纪要标准模板

本文档提供标准化的会议纪要模板，确保输出格式一致、信息完整。

## 简化模板（小型会议）

适用于参与人数少、议题简单的会议：

```markdown
# YYYY-MM-DD 会议主题

**参与者**: [姓名1]、[姓名2]、[姓名3]

## 核心决议

1. [决议1]
2. [决议2]

## 行动项

- [ ] [任务1] - [@负责人] - [截止日期]
- [ ] [任务2] - [@负责人] - [截止日期]

## 下次会议

[时间] - [议题]
```

## 模板使用指南

### 何时使用标准模板

- 正式会议（董事会、管理层会议）
- 跨部门协作会议
- 项目启动/里程碑会议
- 需要存档的重要会议

### 何时使用简化模板

- 日常站会
- 小组讨论
- 临时沟通会议
- 快速决策会议

### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 会议主题 | ✅ | 简洁明确，不超过20字 |
| 会议时间 | ✅ | 格式：YYYY-MM-DD HH:MM |
| 参与者 | ✅ | 至少包含姓名，职位可选 |
| 核心决议 | ⚠️ | 如有决议必填 |
| 行动项 | ⚠️ | 如有待办必填 |
| 讨论要点 | ⚪ | 可选，但建议填写 |
| 下次会议 | ⚪ | 如已确定则填写 |

### 优先级定义

| 优先级 | 含义 | 截止时间 | 示例 |
|--------|------|----------|------|
| **P0** | 紧急且重要 | 本周内 | 修复线上bug、紧急客户需求 |
| **P1** | 重要不紧急 | 2周内 | 功能开发、文档更新 |
| **P2** | 常规任务 | 1个月内 | 优化改进、技术债务 |

## 注意事项

1. **时间格式统一**: 使用 `YYYY-MM-DD HH:MM` 格式
2. **负责人标注**: 使用 `@姓名` 格式，便于搜索
3. **优先级一致**: 全文使用统一的 P0/P1/P2 标准
4. **状态更新**: 行动项状态应及时更新
5. **原始记录**: 始终保留原始记录路径，便于追溯
